3- Copy all folders under the Guns folder to the Strike Fighters 2\Objects\Guns folder

4- Copy all folders under the Pilots folder to the Strike Fighters 2\Objects\Pilots folder

5- Copy all folders under the Weapons folder to the Strike Fighters 2\Objects\Weapons folder
